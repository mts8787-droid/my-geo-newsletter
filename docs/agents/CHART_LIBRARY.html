Chart Library — 차트 분류 카탈로그

차트 양식별 분류 코드 + 예시 SVG + 사용 위치

분류 체계란? 차트 양식마다 짧은 코드 (예: L-1, BP-1) 를 부여한 호출 시스템.

왜 쓰나? Skill이 차트를 그릴 때 어떤 양식인지 명확히 지정 — "L-1 형태로 라인 차트 만들어줘" 같이 코드만 알려주면 정확한 양식으로 그려짐.

  • L — 라인 (시계열 / 트렌드)
  • B — 바 스택 (누적 / part-to-whole)
  • V — 그룹 막대 vbar (카테고리별 N 개)
  • HB — 가로 막대 (Top N 랭킹)
  • M — 콤보 (Bar + Line 결합)
  • BP — 범프 (순위 변화)
  • MT — 미니 트렌드 (이메일 호환)
  • D — 도넛 / 파이 (비중)
  • R — 레이더 (다차원 metric)
  • BU — 버블 / 4분면 (X/Y 산점)
  • T — 툴팁 (직교 — 어느 차트에든 추가)

카테고리 뒤 숫자는 양식 번호 (같은 카테고리 안 변형 구분). 예: L-1 기본 라인 / L-5 베이스라인 fade 라인.

L — 라인 차트

Line — 시계열 / 트렌드

L-1
Basic Line
기본 직선 라인 — 가장 단순. 카드 미니 트렌드의 기본.
사용: 대시보드 카드 미니 트렌드 (svgLine)
Skill 호출 예시: "L-1 차트 그려줘"
L-2
Interpolation (곡선)
Q/C Bezier 곡선 연결. 본 저장소 미사용 (시각 일관성 위해 직선만).
사용: 미사용
Skill 호출 예시: "L-2 차트 그려줘"
L-3
Multi-axis
좌/우 Y축 분리 — 단위 다른 두 series.
사용: 미사용 (도입 후보: KPI % + 호출 수)
Skill 호출 예시: "L-3 차트 그려줘"
L-4
Point Styling
점 모양 / 크기 / 색 커스터마이즈. 베이스라인 시작점만 사용.
사용: 베이스라인 시작점 (검정 fill + 컬러 stroke)
Skill 호출 예시: "L-4 차트 그려줘"
L-5
Segments
구간별 다른 시각 (색 / 두께 / dash). fadeBeforeIdx 가 이 컨셉.
사용: 베이스라인 fade (pre 회색 / post 컬러)
Skill 호출 예시: "L-5 차트 그려줘"
L-6
Stepped
계단형 (직각 연결). step-change 데이터 시 적합.
사용: 미사용 (도입 후보: 분기 데이터, 정책 변경 시점)
Skill 호출 예시: "L-6 차트 그려줘"
L-7
Styling
색 / 두께 / dash 패턴. dashed 베이스라인 수직선 / 비교군 회색.
사용: dashed 베이스라인, 비교군 라인 회색
Skill 호출 예시: "L-7 차트 그려줘"

B — 바 차트 (스택)

Bar — 카테고리 누적 (Chart.js 표준)

B-1
Stacked Bar
단일 스택 — 카테고리당 1 컬럼 누적 (part-to-whole).
사용: 미사용 (도입 후보: BU별 누적, Citation type 누적)
Skill 호출 예시: "B-1 차트 그려줘"
B-2
Stacked Groups
카테고리당 N stack 가로 나란히 (그룹 간 비교).
사용: 미사용 (도입 후보: KR vs EN, LG vs Samsung 누적)
Skill 호출 예시: "B-2 차트 그려줘"

V — 그룹 막대 (본 저장소)

Vbar — 카테고리당 N 개 막대 (그룹)

V-1
그룹 막대 (vbar)
본 저장소 C-05. 카테고리당 N 개 막대 가로 나란히. _citVBar 헬퍼 + scale 파라미터.
사용: dashboard / citation 의 카테고리별 vbar (Reddit/YouTube 등)
Skill 호출 예시: "V-1 차트 그려줘"

HB — 가로 막대 (Top N)

Horizontal Bar — 랭킹 / Top N

HB-1
가로 막대 (Top N)
본 저장소 C-06. 도메인 / 채널 랭킹용. Top N 정렬 + 비중 표시.
사용: citation 의 도메인 Top N 랭킹
Skill 호출 예시: "HB-1 차트 그려줘"

M — 콤보 차트

Mixed — Bar + Line 결합

M-1
Combo Bar+Line
Bar (절댓값) + Line (추세) 단일 Y축. 도입 후보: KPI 값 + 증가율.
사용: 미사용
Skill 호출 예시: "M-1 차트 그려줘"
M-2
Stacked Bar+Line
누적 막대 + 합계 추세 라인 overlay. 도입 후보: BU 누적 + 평균 추세.
사용: 미사용
Skill 호출 예시: "M-2 차트 그려줘"

BP — 범프 차트

Bump — 순위 변화 시각화

BP-1
범프 차트 (Bump)
본 저장소 C-09. 순위 변화 시각화. viewBox 비율 + linecap=round + 폰트 +3pt + 하단 여백 +20px.
사용: citation 의 카테고리 / 도메인 월별 랭킹 변화 (bumpChartSvg)
Skill 호출 예시: "BP-1 차트 그려줘"
